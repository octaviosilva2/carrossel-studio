# Sessão S4 — Export — 2026-07-01

## Objetivo
Fechar o ciclo do produto: transformar o carrossel montado no editor em arquivos prontos para o
Instagram. Três entregas: (1) PNG de **todos** os slides, cada um exatamente **1080×1350**;
(2) download como **ZIP** + download individual do slide selecionado; (3) nitidez garantida
(pixelRatio) e nomeação ordenada dos arquivos. Conduzida pela esteira `dev-agents`.

## Decisões-chave
### Produto (CEO, gate)
- **Nitidez vs dimensão:** manter **1080×1350 exatos** com `pixelRatio: 1`. O nó já renderiza em
  tamanho físico real, então 1× é nitidez máxima na resolução-alvo do Instagram; 2× quebraria o
  "exatamente 1080×1350". Sem export em 2× / formatos alternativos.
- **Gatilho:** só no `/editor` com o carrossel aberto. Export a partir da lista `/carousels` fica
  fora de escopo (sessão futura).
- **Nomeação:** `slide-01.png`, `slide-02.png`… (zero-pad p/ ordenação correta acima de 9 slides);
  ZIP `<titulo-slug>.zip`, sem título → `carrossel.zip`.
- **Slide vazio** exporta em branco (render é determinístico); bloqueio só com 0 slides.

### Técnica (CTO)
- **Tainted canvas resolvido** (pendência herdada da S3): a imagem cross-origin do Blob é convertida
  para **data-URL antes do canvas** (`toExportSafeUrl`: fetch → blob → FileReader), **sem tocar o
  `<Slide>` compartilhado** — o preview não regride.
- **Fetch direto** ao CDN do Blob (GATE 2, YAGNI). O proxy same-origin `/api/blob/proxy` ficou
  **desenhado como plano B, não implementado** — só entra se o CORS do CDN reprovar no smoke.
- **Captura sob demanda:** `<ExportCapture>` monta os N `<Slide>` 1080×1350 reais off-screen só no
  clique de export (não a cada tecla), padrão do `render-test`. Captura **sequencial** (não paralela)
  para não estourar memória em carrosséis longos.
- **jszip** (client-side) para montar o ZIP → Blob → download via `URL.createObjectURL`.

- `npm run type-check`: **limpo**. `npm run build`: **compilou** (6/6 páginas).
- `npm test`: **171/171** (137 baseline sem regressão + 34 novos) + **1 skip** (guardião dimensional
