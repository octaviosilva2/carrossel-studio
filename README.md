# Carrossel Studio

Plataforma web para gerar carrosséis estilo Twitter/X (modelo Octavio) prontos para o Instagram.

- **Modelo:** done-for-you (uma instância configurada por cliente).
- **Duas portas de entrada:** com IA (montadora) e manual (passo a passo) — mesmo motor.
- **Saída:** PNGs 1080×1350, tema claro ou escuro.
